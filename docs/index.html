Source Mapper

SourceMapper is code generator for mappings based on attributes defined in interfaces or abstrat classes. It is based on .NetCore Source Generators engine and generates classes for mappers during build time of you project. That is the main difference between SourceMapper and Automapper: you can see, reuse or inherit from generated code after app build process.
